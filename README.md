# 🚌 TravelTelegramBot

TravelTelegramBot — это интеллектуальный Telegram-бот на C#, предназначенный для отслеживания расписания общественного транспорта. Бот автоматически определяет тип дня (будни/выходные), рассчитывает время до ближайшего рейса и предоставляет информацию в удобном для чтения виде.

 ✨ Основные функции

•   📅 Умное расписание: Автоматическое переключение между будним и выходным графиком.
•   🕒 Расчет «Ближайшего»: Бот показывает, через сколько минут приедет следующий автобус.
•   🌍 Timezone Management: Корректная работа на серверах с любым системным временем (настроено на UTC+3).
•   📱 Оптимизированный UX: Специальная верстка кнопок для удобного нажатия на современных смартфонах.
•   🎨 HTML-форматирование: Красивые, структурированные ответы с использованием эмодзи и моноширинных шрифтов для времени.

 🛠 Технологический стек

•   Язык: C# (.NET 10)
•   Библиотека: Telegram.Bot (https://github.com/TelegramBots/Telegram.Bot)
•   Логика парсинга: Composite Design Pattern для обработки контента.
•   База данных: JSON-файл

 🚀 Быстрый запуск

 Предварительные требования
•   .NET 10.0 SDK
•   Telegram Bot Token (полученный у (@BotFather (https://t.me/botfather))

 Установка
1. Клонируйте репозиторий:
   
   git clone https://github.com/ваш-логин/TravelTelegramBot.git
   
2. Перейдите в папку проекта:
   
   cd TravelTelegramBot
   
3. Установите токен бота в конфигурации (или через переменные окружения):
   
   export BOT_TOKEN='ваш_токен_здесь'
   
4. Запустите проект:
   
   dotnet run
   
 📖 Пример работы

При запросе расписания бот выдает сообщение в следующем формате:
> 🚌 Маршрут 101
> 💼 СЕГОДНЯ БУДНИЙ ДЕНЬ
> 
> РАСПИСАНИЕ БУДНЕГО ДНЯ
> 📍 Станция А ➡️ Станция Б
> ✅ Ближайший: 14:30 (через 15 мин)
> 🕒 08:00, 10:00, 12:00, 14:30, 17:00
